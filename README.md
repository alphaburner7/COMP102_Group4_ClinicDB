# Clinic Database System – COMP102 Final Project

## 📌 Project Overview
This project implements a **Public Health Clinic Records System** using MySQL (XAMPP).  
It is aligned with **SDG 3: Good Health and Well-being** and demonstrates database design, implementation, and management skills.

## 🧩 Features
- Refined Physical Data Model (Patients, Doctors, Appointments, Treatments, Payments)  
- Database creation and table implementation with PK/FK constraints  
- Data manipulation (INSERT, UPDATE, DELETE) with meaningful sample data  
- SQL queries (filtering, sorting, aggregates, LIMIT)  
- Basic user management (user creation, privileges, password changes)  

## 📂 Files Included
- `clinic_db.sql` → Database creation, tables, sample data, queries  
- `Database_Group4#A#A.pdf` → Project report  
- Screenshots → Database creation, queries, and user management  

## 🚀 How to Run
1. Open **XAMPP** and start Apache + MySQL.  
2. Go to `http://localhost/phpmyadmin`.  
3. Import `clinic_db.sql` into phpMyAdmin.  
4. Run queries and test functionality.
